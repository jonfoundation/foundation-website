# Foundation Website Launch Package

Updated homepage using the new Foundation logo system with navy, sand, sage, and warm neutral styling.

## Deploy to GitHub and Cloudflare Pages
1. Unzip this package on your personal computer.
2. Create a GitHub repository named `foundation-website`.
3. Upload the **contents** of this folder so `index.html` is at the repository root.
4. In Cloudflare: **Workers & Pages → Create → Pages → Connect to Git**.
5. Select the repository.
6. Framework preset: **None**
7. Build command: leave blank
8. Build output directory: `/`
9. Deploy.
10. Add `foundationsd.co` as the custom domain.
